form表单知识点

1、表单标签组

那么，表单该怎么用HTML标签呢？是用form <form name="ff" id="zhuce" method="POST" action="data.php" > </form> 在表单FORM中，将包含下列几种子标签：
  • 文本框 <input type="text" id="email" value="姓名" />
  • 按 钮 <input type="button" id="anniu" value="点我" />
  • 提 交 <input type="submit" id="tijiao" value="注册" />
  • 重 置 <input type="reset" id="start" value="重置" />
  • 单选按钮： <input type="radio" name="danxuan" value="male" />男人 <input type="radio" name="danxuan" value="female" />女人 <input type="radio" name="danxuan" value="secret" />保密
  • 多选按钮：（也叫复选按钮） <input type="checkbox" name="tc" value="read" />阅读 <input type="checkbox" name="tc" value="play" />体育 <input type="checkbox" name="tc" value="chat" />聊天
  • 菜单： <select name="kecheng" id="lesson" > <option>历史</option> <option>地理</option> <option>数学</option> </select>
  • 文本区域：<textarea name="neirong" id="content" > 这里输入的文字显示在文本区域内容 </textarea>

2、JS与表单或表单组标签的交互

document.write(“内容"); //输出内容

回顾JS函数的定义方法：

function 函数名(这里写参数，可不设定，多个用逗号隔开){ //中间写函数的执行程序 /*多行注释，中间不运行*/ var x=10; }

JS中的函数如果不调用，它不会执行。必须由网页事件行为驱动才可以。

常见的网页事件行为包括：

  • 1、 单击：click
  • 2、 双击：dblclick
  • 3、 鼠标放上去：mouseover
  • 4、 鼠标放下来：mouseout
  • 5、 开始敲击键盘：keydown
  • 6、 键盘抬起时：keyup
  • ……

on+事件行为代码来驱动函数程序运行。

【举例】 <script language="javascript"> <!— function clickme(){ alert("哈喽，你好啊！"); } --> </script> …… <input type="button" id="cat" onclick="clickme()">

Clickme()函数中的参数可变换设定，但一旦设定将必须按结构调用。

Clickme(x,y,a) clickme(1,5,3)

函数从id中获取想要的信息。

document.getElementById(“id名称").value

如果用到form，就以name设定的名称，使用form.name.value

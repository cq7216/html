点击时弹出一个对话框

1、知识点

$(document).ready(function(){//功能});表示文档加载完后执行某功能，此用法依赖于jquery.js

$('a').click(function(){//功能});表示点击a标签时执行某功能

$('.button').click(function(){//功能});表示点击calss=button元素时执行某功能

$('input:text').click(function(){//功能});表示点击文本框标签时执行某功能

$('input:text:first').click(function(){//功能});表示点击第一个文本框标签时执行某功能

$('input:text:eq(2)').click(function(){//功能});表示点击第三个文本框标签时执行某功能，0为第一个，1为第二个，2为第三个

$('input:text:even').click(function(){//功能});表示点击第奇数个文本框标签时执行某功能

$('input:text:odd').click(function(){//功能});表示点击第偶数个文本框标签时执行某功能

$('input:text:last').click(function(){//功能});表示点击最后一个文本框标签时执行某功能

$('input:checkbox[name="dd"]').click(function(){//功能});表示点击name=dd的复选框标签时执行某功能

2、用法一

在head标签中添加下面代码

<!-- 后盾网写法开始 --> <script type="text/javascript" src="js/jquery-1.6.2.min.js"></script> <script type="text/javascript"> $(document).ready(function(){ $('a').click(function(){ alert("这是a弹出的内容"); }); $('.button').click(function(){ alert(document.getElementById("anniu").value);//弹出id=anniu的元素的value值 }); $('input:text:first').click(function(){ alert("这是第一个文本框弹出的内容"); }); $('input:text:eq(2)').click(function(){ alert("这是第三个文本框弹出的内容");//从0开始，2表示第三个 }); $('input:text:last').click(function(){ alert("这是最后一个文本框弹出的内容"); }); $('input:text:odd').click(function(){ alert("这是第偶数个文本框弹出的内容");//奇数用even表示，偶数用odd }); $('input:checkbox[name="dd"]').click(function(){ alert("这是box弹出的内容"); }); $('#dd').click(function(){ alert("这是div弹出的内容"); }); }); </script> <!-- 后盾网写法结束 -->

下面是body标签中的内容

<a href="#">点击弹出一个对话框</a> <input type="button" class="button" value="按钮"> <input type="text" class="text"> <input type="text" class="text"> <input type="text" class="text"> <input type="text" class="text"> <input type="text" class="text"> <input type="checkbox" name="dd" id=""> <div style="width:100px;height:100px;background-color:#eee;" id="dd"></div>

:first表示第一个，:last表示最后一个，:even表示第奇数个，:odd表示第偶数个，:eq(0)表示第一个，:gt(0)表示大于第0个，:eq(n)表示第n-1个，[name="dd"]表示name为dd的标签

4、用法二

也可用此方法（老杜写法）

<!-- 老杜写法开始 --> <script language="javascript"> <!-- function clickme(){ alert("这是老杜点击弹出的内容"); //document.getElementById("id名称").value函数从id中获取想要的信息 } --> </script> <!-- 老杜写法结束 -->

下面是body标签中的内容

<input type="button" value="单击按钮" onclick="clickme()"> <input type="button" value="双击按钮" ondblclick="clickme()"> <input type="button" value="鼠标放上" onmouseover="clickme()"> <input type="button" value="鼠标移开" onmouseout="clickme()">

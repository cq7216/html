phpcmsV9添加多文件上传字段并调用

一、操作方法

有时想给文章增添一个或多个附件，方便下载，比如给文章附上word原文，那么如何修改文章模型，在发布文章的时候上传多个附件，并在内容模板中调用呢？

  • 1、依次打开登陆后台-内容-模型管理-文章模型-字段管理-添加字段
  • 2、字段类型选择多文件上传--字段名起“word”，起好别名，允许同时上传数量改为1，点击提交
  • 3、返回文章模型字段管理，排序
  • 4、管理内容--编辑文章时上传word文档

二、调用方式

在show.html模板中添加：

附件地址为： {pc:get sql="select * from v9_news_data where id=$id order by id asc " } {loop $data $r} {php $xiazai = string2array($r['word']);} {php echo $xiazai[0][fileurl]; } {/loop} {/pc}

若想直接调用，可直接使用下列代码

{loop $word $r} {$r} {/loop}
